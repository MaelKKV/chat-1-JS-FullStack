# Chat 1 JS FullStack

It is a simple chat that was built with HTML5, CSS3, BootStrap 4, and javascript technologies, for the frontend, backend and database.

![](screenshots/Screenshot Chat JS FullStack.png)

# Used Technologies
- Nodejs
  - Express
  - Socket.io
  - Nodemon
- Mongodb
  - Mongoose
